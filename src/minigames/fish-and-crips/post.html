Gang warfare has reached an all-time high in the Pacific Ocean. Which side will you choose?

Controls:

Red:

  • Move: WASD
  • Shoot: Left Control

Blue:

  • Move: Arrow Keys
  • Shoot: Right Control

Features

  • 2-player competitive multiplayer
  • Fish with guns
  • Scoreboard

This game can only be played on computers - unfortunately, it does not support mobile devices.
