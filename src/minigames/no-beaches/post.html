Overview

Beach your opponents! The last boat floating wins.

Controls

WASD

  • Move: WASD
  • Boost: Left Shift

Arrows

  • Move: Arrows
  • Boost: Right Shift

Controller 1

  • Move: Joystick
  • Boost: Primary Button

Controller 2

  • Move: Joystick
  • Boost: Primary Button

Features

  • Local multiplayer
  • AI opponents
  • Controller and keyboard support
  • Procedurally generated map
  • Up to 4 players
  • Beeg Boost mode
  • Explosions
  • b u o y

This game can only be played on computers - unfortunately, it does not support mobile devices.
