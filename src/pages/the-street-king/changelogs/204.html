September 16, 2020

Another minor update is here to improve the game.

Additions

  • Added label in the main menu that shows what version of the game is running
  • Added automatic cache clearing
  • Added higher quality road textures

Changes

  • Render distance for traffic cars has been increased even on the lowest quality settings to avoid them popping up suddenly
  • Very minor changes to how the main menu buttons respond to controller inputs

Bug Fixes

  • Fixed teleporting racers in singleplayer
  • Fixed police car counter being stuck at 0 in chases
  • Fixed large engines clipping through the hood of two cars
  • Fixed map icon scaling being delayed by one frame from the actual camera zoom
