January 22, 2021

Minor Update 2.31 is here with lots of performance improvements, bug fixes, and more!

Additions

  • Added option to remove ads completely through an in-app purchase
  • Added ability for polls to show up in the announcments screen
  • Added support for multithreaded rendering, which improves performance
  • Added incremental garbage collection to improve performance
  • Added support for Vulkan

Changes

  • Improved some pathing areas of the GPS system
  • Changed shader loading logic to improve performance
  • Other performance improvements

Bug Fixes

  • Fixed inverted up/down direction controls in the map menu.
  • Fixed decals not showing up correctly on some cars with popup headlights.
  • Fixed police cars spawning clipped inside of the barn walls in the final race.
  • Fixed some misplaced palm trees in the beach
  • Fixed the ban query and multiplayer support buttons not autofilling the contact email address
  • Fixed issues with the Atlas Bridge loading in delayed when traveling towards Atlas City
  • Fixed bug where the ranks of very poorly performing cars (tier 0) would be too high in the garage
  • Fixed bug where the glass in hoods would change to carbon fiber as well when using a carbon fiber hood
  • Fixed bug where the sunroof cover of cars would change to carbon fiber as well when using a carbon fiber roof scoop.
