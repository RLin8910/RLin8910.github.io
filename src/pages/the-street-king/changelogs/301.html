March 28, 2022

Version 3.01 fixes some bugs.

Bug Fixes

  • Fixed transmission reverting to automatic after an online race with drag physics even if the player had their controls set to manual
  • Fixed race hosting ending if any player left the race, even if it was not the host
  • Fixed bug where purchasing an engine, returning to stock engine, and then selecting the previous engine type again would result in no engine being selected
  • [iOS] Fixed rendering issues when light mode system theme is selected
  • [iOS] Fixed game audio not muting when advertisments play
