November 18, 2020

Update 2.11 fixes some bugs.

Changes

  • Minor advertising changes
  • Balancing changes to the amount of money earned per race
  • Improved how the game detects what quality level to use

Bug Fixes

  • Fixed a bug where pressing the reset button in the test zone before driving around would cause the car to reset to the wrong spot
  • Fixed controller input not working for the "Auto-Backup" toggle button in the cloud save menu
  • Fixed the "x" icon not disappearing in the "Auto-Backup" toggle button in the cloud save menu when no controller is connected
  • Fixed an issue with the collision box of the canyon rocks near General Road
  • Fixed an issue with the collision box of the fences near Creekside Road
  • Fixed a bug where quitting a race right before totalling your car could cause some issues with the camera and menus
  • Fixed a bug where quitting the game and coming back would allow you to jump back to the garage and avoid getting busted
  • Fixed a bug where flipping upsidedown in the tutorial while a police car is nearby would softlock the game
  • Fixed a bug where the engine of one of the cars did not move with the body.
  • Fixed "Grayscale" being spelled as "GrayScale" in the filters menu when taking screenshots
  • Fixed the "Buy Money" window being too large to fit in certain screens in the showroom
  • Fixed bug where skipping the intro too quickly when opening the app for the first time could result in the tutorial being skipped
