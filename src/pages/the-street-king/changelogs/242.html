Update 2.42 fixes some bugs.

Additions

  • The finish line icon is now shown in multiplayer races

Changes

  • Concrete barrier roadblocks are no longer allowed to spawn during multiplayer races
  • Improved performance with some devices

Bug Fixes

  • Fixed issues with the pre-race positioning system in multiplayer that could cause the player to clip out of bounds
  • Fixed bug where selecting an event and then leaving without starting it would deactivate the current GPS route
  • Fixed bug where wrecking during a drag race would not cause the player to lose the bet
