September 2, 2020

Version 2.02 has been released with a few small changes.

Additions

  • Added announcements menu, which can be changed in-game without requiring an update
  • Added custom tag and icon for me to use when playing online
  • Map icons now scale up when zooming out to maintain visibility

Changes

  • Since I now have a custom tag, players can no longer have yellow usernames. Yellow usernames are now reserved only for me.
  • Some map icons have had outlines added to them to improve visibility.

Bug Fixes

  • Fixed error 314
  • Fixed a few camera placement issues in some of the cars
  • Fixed "rank 606" bug where you could get your car to show up with an abnormally high rank in multiplayer
